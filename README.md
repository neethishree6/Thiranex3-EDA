# COVID-19 Exploratory Data Analysis (EDA) Project

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on COVID-19 datasets using Python. The main objective is to uncover patterns, trends, correlations, and insights from real-world COVID-19 data through statistical summaries and visualizations.

The project demonstrates data preprocessing, analysis, and storytelling using various visualization techniques.

---

## Objectives

* Analyze COVID-19 datasets
* Identify trends and patterns
* Perform statistical analysis
* Detect correlations between variables
* Create meaningful visualizations
* Present findings in a structured report

---

## Datasets Used

The following datasets were used in this project:

1. `covid_19_india.csv`
2. `covid_vaccine_statewise.csv`
3. `StatewiseTestingDetails.csv`

* Confirmed COVID-19 cases increased rapidly over time.
* Strong correlation exists between confirmed cases and deaths.
* Certain states contributed significantly more cases than others.
* Vaccination rates varied widely across states.
* Outliers were identified in confirmed case distributions.
